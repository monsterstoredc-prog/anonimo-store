# Anonimo Store

Loja de packs com painel admin e integração pronta para Sunize (PIX).  
Crie packs pelo /admin, cliente compra na página inicial. Webhook: /webhook/sunize

## Rodando local
- `npm install`
- `npm run migrate`
- `npm start`

## Variáveis de ambiente
- ADMIN_PASSWORD
- BASE_URL
- SUNIZE_API_KEY
